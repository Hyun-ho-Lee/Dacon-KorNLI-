# Dacon KorNLI

**1. train_data.csv**

├ Index : train data index

├ Premise : 실제 Text

├ Hypothesis : 가설 Text

└ Label : **참(Entailment)** 또는 **거짓(Contradiction)** 또는 **중립(Neutral)**

### **2. test_data.csv**

├ Index : test data index

├ Premise : 실제 Text

├Hypothesis : 가설 Text

└Label : 추론해야 하는 Label 값
